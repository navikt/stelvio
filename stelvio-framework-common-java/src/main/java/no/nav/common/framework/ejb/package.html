Enterprise Java Beans (EJB) Framework. This package contains classes and interfaces to support rapid development of EJB functionality, such as:
  • JNDI Lookup - EJBLookupHelper
  • Service Delegate - AbstractServiceDelegate
  • Service Facade - AbstractServiceDelegate
  • Service Location - RemoteServiceLocator
  • Service Registry - RemoteServiceLocator, RemoteServiceDescription
  • Service Adapter - SessionAdapter
